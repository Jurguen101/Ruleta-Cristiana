# 🎡 Ruleta Bíblica Interactiva en Python

¡Bienvenido al proyecto de la **Ruleta Bíblica**! Este es un juego interactivo de escritorio desarrollado completamente en Python utilizando la librería `Tkinter`. Es ideal para eventos de iglesia, grupos de jóvenes, clases de escuela dominical o simplemente para pasar un rato divertido aprendiendo en familia.

## ✨ Características Principales

* **Ruleta Gráfica y Dinámica:** Una ruleta colorida que gira con físicas reales (aceleración y fricción) y calcula automáticamente en qué sección cayó.
* **Autoajustable:** Si una de las 10 secciones se queda sin sus 5 preguntas, esa sección es eliminada y la ruleta recalcula sus tamaños para rellenar los 360 grados.
* **Temporizador Integrado:** Cada pregunta activa un reloj en pantalla de 10 segundos antes de revelar la respuesta.
* **Animaciones y Efectos:** * Un hermoso fondo de "cielo estrellado" en movimiento continuo.
  * ¡Animación de confeti con físicas de gravedad cayendo por la pantalla al revelar las respuestas!
* **Panel de Estadísticas:** Un panel lateral en tiempo real que muestra cuántas preguntas le quedan a cada sección antes de agotarse.
* **Modo Pantalla Completa:** El juego se ejecuta en pantalla completa para una experiencia inmersiva tipo programa de televisión.

## 🛠️ Requisitos

Este proyecto está construido con la biblioteca estándar de Python, por lo que **no necesitas instalar librerías externas** pesadas. 

Solo necesitas tener instalado:
* **Python 3.x** (Puedes descargarlo desde [python.org](https://www.python.org/downloads/))

*(Las librerías utilizadas como `tkinter`, `random` y `math` ya vienen preinstaladas con Python).*
